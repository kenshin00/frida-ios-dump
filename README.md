# frida-ios-dump
pull decrypted ipa from jailbreak device

### Usage

## 1. install [frida](http://www.frida.re/) on device and mac

## 2. iproxy 2222 2222

## 3. ./dump.py bundle_id_or_app_name proc_name


congratulations!!! You've got a decrypted ipa file.
