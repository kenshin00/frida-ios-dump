# frida-ios-dump
dump decrypted ipa from jailbroken devices

### Usage

## 1. install [frida](http://www.frida.re/) on device and mac

## 2. iproxy 2222 2222

## 3. ./dump.py bundle_id_or_app_name proc_name

## 4. use frida-ps to find proc_name
